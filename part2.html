Del 2: JavaScript Oppgaveliste

Bygg en interaktiv oppgaveliste med ren JavaScript

⏱️ 15:00
1

HTML Grunnstruktur

Bygg grunnstrukturen for oppgavelisten med input, knapp og liste.

💡 Oppgave: I HTML, inni <div class="container">, legg til:
<h1>✅ Min Oppgaveliste</h1>
<div class="input-gruppe">
• <input type="text" id="oppgave-input" placeholder="Skriv en oppgave...">
• <button id="legg-til-btn">Legg til</button>
</div>
<ul id="oppgave-liste"></ul>
2

CSS Styling

Gjør oppgavelisten pen med CSS — bruk flexbox, farger og spacing.

💡 Oppgave: I CSS, style applikasjonen:
body { font-family: Arial; background: #1a1a2e; color: #eee; display: flex; justify-content: center; padding: 40px; }
.container { background: #16213e; padding: 30px; border-radius: 12px; width: 500px; }
#oppgave-input { flex: 1; padding: 12px; border: none; border-radius: 6px; }
#legg-til-btn { padding: 12px 20px; background: #6366f1; color: white; border: none; border-radius: 6px; cursor: pointer; }
.input-gruppe { display: flex; gap: 10px; margin-bottom: 20px; }
3

Legg til oppgaver (JS)

Bruk JavaScript til å lese input og legge til nye oppgaver i listen.

💡 Oppgave: I script.js:
const input = document.getElementById('oppgave-input');
const btn = document.getElementById('legg-til-btn');
const liste = document.getElementById('oppgave-liste');

btn.addEventListener('click', function() {
  if (input.value.trim() === '') return;
  const li = document.createElement('li');
  li.textContent = input.value;
  liste.appendChild(li);
  input.value = '';
});
4

Slett oppgaver (JS)

Legg til en slett-knapp (×) på hver oppgave som fjerner den fra listen.

💡 Oppgave: Legg til en slett-knapp i funksjonen din:
const slettBtn = document.createElement('button');
slettBtn.textContent = '×';
slettBtn.className = 'slett-btn';
slettBtn.addEventListener('click', function(e) {
  e.stopPropagation();
  li.remove();
});
li.appendChild(slettBtn);
5

Merk som ferdig (JS)

Klikk på en oppgave for å markere den som ferdig med gjennomstreking.

💡 Oppgave: Legg til click-event på hvert li-element:
li.addEventListener('click', function() {
  this.classList.toggle('ferdig');
});

I CSS, legg til:
.ferdig { text-decoration: line-through; opacity: 0.5; }
6

Oppgaveteller (JS)

Vis en teller som viser antall oppgaver og hvor mange som er fullført.

💡 Oppgave: Lag en teller-funksjon:
function oppdaterTeller() {
  const alle = document.querySelectorAll('#oppgave-liste li');
  const ferdige = document.querySelectorAll('#oppgave-liste .ferdig');
  document.getElementById('teller').textContent =
    alle.length + ' oppgaver (' + ferdige.length + ' fullført)';
}
Kall oppdaterTeller() etter legg til, slett og ferdig-klikk.
7

Filtrering (JS)

Legg til filterknapper for å vise alle, aktive eller fullførte oppgaver.

💡 Oppgave: Legg til filterknapper i HTML:
<div class="filter-knapper">
  <button class="aktiv" data-filter="alle">Alle</button>
  <button data-filter="aktive">Aktive</button>
  <button data-filter="fullforte">Fullførte</button>
</div>

I JavaScript, gå gjennom alle li og sett style.display basert på filter.
8

LocalStorage (JS)

Lagre oppgavene i nettleseren slik at de overlever en sideoppdatering.

💡 Oppgave: Lagre oppgaver:
function lagreOppgaver() {
  const oppgaver = [];
  document.querySelectorAll('#oppgave-liste li').forEach(li => {
    oppgaver.push({ tekst: li.firstChild.textContent, ferdig: li.classList.contains('ferdig') });
  });
  localStorage.setItem('oppgaver', JSON.stringify(oppgaver));
}

Last oppgaver ved start med JSON.parse(localStorage.getItem('oppgaver'))

🌲 DOM Forklart

DOM er en trestruktur som representerer HTML-dokumentet. JavaScript kan traversere og endre denne strukturen for å oppdatere siden.

🎯 Event Flow

Events "bobler" oppover i DOM-treet. Du kan lytte på events på hvilken som helst node i hierarkiet.
